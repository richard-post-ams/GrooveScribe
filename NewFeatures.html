Shipped Features

4
Web Audio API Drum Sampler — Better Sound Quality
Replaces the thin MIDI.js soundfont with higher-quality OGG drum samples loaded via the Web Audio API. Covers kick, snare, hi-hat (open/closed/foot), toms, crash, ride, and rimshot using the FatBoy GM soundfont (CC0). Falls back silently to the original MIDI.js playback if the sampler is not yet loaded.
Shipped #46f6f28 #f34db2e

5
Mobile Responsive Layout
Makes GrooveScribe usable on phones and tablets. Removes the hard 800px min-width, stacks the left nav horizontally on small screens, adds 44px+ tap targets on all note buttons, enables horizontal scroll on the note grid, prevents iOS input zoom, and adds touch-action manipulation to stop double-tap zoom. Three breakpoints: tablet (≤900px), phone (≤480px), and coarse-pointer (touch) devices.
Shipped #c1c84e2 #63615f5

10
Local Groove Library with Save / Load
Adds a localStorage-based groove library panel accessible via the new LIBRARY button in the bottom toolbar. Save any groove with a name and comma-separated tags. Search by name or tag, load with one click, delete with confirmation. Stores as many grooves as localStorage allows. Works fully offline with no server needed.

Planned Features

1
Fix 10-Measure Cap
The app hard-stops at 10 measures. Remove the hard-coded measure limit in groove_writer.js so longer grooves, fill sequences, and full song charts can be written.
Planned
2
Fix Broken Share Links (Firebase Deprecation)
Firebase Dynamic Links was deprecated August 2025. The Share button is currently broken. Replace with a self-hosted redirect or a lightweight URL shortener alternative.
Planned
3
PWA / Offline-First Support
Add a service worker and web app manifest so GrooveScribe works offline in rehearsal spaces and can be installed to the home screen. The fully client-side architecture already supports this.
Planned
6
Loop Section / A-B Repeat for Practice
Add a visual A-B loop selector so you can mark any subset of measures and drill just that section. Standard feature in every serious practice tool (Transcribe!, Amazing Slow Downer). Currently the player only loops the entire groove from start to end.
Planned
7
Note Velocity / Ghost Notes / Accents
Every hit is currently binary: on or off. Add three velocity levels per note (ghost, normal, accent) via right-click or long-press. The ABC notation renderer already supports dynamics via !p!, !mf!, !f! decorators.
Planned
8
Multi-Step Undo / Redo Stack
The current UI has a single-step Undo button. Replace with a full multi-step undo/redo stack (Ctrl+Z / Ctrl+Y) so no work is lost from a single misclick during complex groove authoring.
Planned
9
Extended Tuplet Support
The UI offers 1/8 and 1/16 triplets but no quintuplets, septuplets, or other tuplets. The abc2svg renderer already supports arbitrary tuplets. The fix is in the writer UI grid logic, not the renderer.
Planned
